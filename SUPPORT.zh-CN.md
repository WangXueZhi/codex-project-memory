# 支持

[English](SUPPORT.md) | 简体中文

## 获取帮助

安装问题、缺陷报告、文档问题和功能建议请提交到
[GitHub Issues](https://github.com/WangXueZhi/codex-project-memory/issues)。

提交前请：

1. 确认使用 Node.js 22.13 或更高版本。
2. 报告源码构建问题时运行 `pnpm check`。
3. 搜索是否已经有相同错误。
4. 删除凭据、私人源码、项目路径、记忆状态和个人信息。

请提供插件版本、操作系统、Codex 使用界面、准确命令、预期结果、实际结果，以及可以公开的
最小复现方式。

## 安全报告

不要公开提交漏洞或敏感数据。请使用
[私有安全公告页面](https://github.com/WangXueZhi/codex-project-memory/security/advisories/new)，
并遵循 [SECURITY.zh-CN.md](SECURITY.zh-CN.md)。

## 支持范围

社区支持按实际能力提供，不承诺回复时间或服务等级。支持范围不包括恢复已经删除的本地记忆、
访问私人项目、管理 Codex 或 OpenAI 账号，以及排查未经支持的分叉版本。
